Permissions Overview

If you would like to know more about permissions in general, Mozilla has a support article about permissions, as well as a guide for assessing the safety of an extension.

Please submit a new issue if you are still concerned about the use of any of the following permissions so the information provided below can be updated to cover common concerns and answer common questions.

Required Permissions

Required Permissions

These are the required browser permissions for YouTube Popout Player.

Access your data for sites in the youtube.com domain
Access your data for sites in the youtube-nocookie.com domain

This permission is used to add the controls (button and context menu) and listen for the keyboard shortcut, which are used to open the popout player, on YouTube videos. This permission is requested for both the primary youtube.com domain and the alternate youtube-nocookie.com domain (which some sites use to embed YouTube videos), as these are the domains where YouTube videos are hosted.

Access browser tabs

The Tabs permission is used to close the original window/tab (when configured) after opening the popout player.
